# serveroptimization

# Getting started
- Clone this repo
- Build / package source
`mvn clean package`

- Start the server
`java -cp target/serveroptimization-1.0-SNAPSHOT.jar edu.ucl.nima.Server 4040`

- Run the client
`java -cp target/serveroptimization-1.0-SNAPSHOT.jar edu.ucl.nima.Client localhost 4040`
